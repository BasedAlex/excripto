## Локальный сервер

Чтобы создать локальный сервер:

Необходимо установить пакет зависимостей через

```
npm install
```

И развернуть сервер локально, через скрипт

```
npm run dev
```

Затем откройте [http://localhost:3000](http://localhost:3000) в браузере.
